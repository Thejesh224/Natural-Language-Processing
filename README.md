# 🤖 AI Sentiment Analysis Chatbot 💬📊

An NLP-based chatbot project built using Python, NLTK, and TextBlob for performing sentiment analysis on user conversations. The chatbot analyzes user input, classifies sentiment, stores interactions with timestamps, and visualizes sentiment trends over time. 🚀

---

# 📌 Project Overview

This project demonstrates the use of **Natural Language Processing (NLP)** techniques to build an interactive chatbot capable of:

✅ Accepting user input in real-time
✅ Performing sentiment analysis on text
✅ Storing conversation history persistently
✅ Tracking sentiment trends using visualization
✅ Using NLP libraries such as NLTK and TextBlob

This project is beginner-friendly and ideal for learning chatbot development and sentiment analysis. 🎯

---

# ✨ Features

🔹 Real-time chatbot interaction
🔹 Sentiment Analysis using:

* TextBlob
* NLTK VADER Sentiment Analyzer

🔹 Positive 😊, Negative 😔, and Neutral 😐 sentiment detection
🔹 Persistent storage of conversations
🔹 Timestamp tracking ⏰
🔹 Data visualization using graphs 📈
🔹 Easy-to-understand implementation



# 🚀 How It Works

1️⃣ User enters a message 💬
2️⃣ Text is processed using NLP 🧠
3️⃣ Sentiment analysis is performed 🔍
4️⃣ Sentiment score is generated 📊
5️⃣ Conversation is stored with timestamp ⏰
